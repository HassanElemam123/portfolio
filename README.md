# Hassan El-Emam Portfolio

Professional React + Tailwind CSS portfolio for **Hassan Mohamed Hassan El-Emam**.

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- Responsive Design

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```bash
http://localhost:5173
```

## Build for Production

```bash
npm run build
npm run preview
```

## Edit Content

Most content is stored at the top of:

```bash
src/App.jsx
```

You can update:

- Name and role
- Email and phone numbers
- GitHub link
- Projects and live demo links
- Skills
- Training and education

## CV File

The CV is included in:

```bash
public/Hassan_El-Emam_Front-End_Developer_CV.pdf
```

The website has a **Download CV** button that points to this file.
